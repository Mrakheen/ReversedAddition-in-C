### Reverse_Adding_C 
#### Done Using C Language

***0 <= Value of Each Array Element <= 9 (single digit array elements)*** 
  <br> 
***0 <= Value of array element <= 9***
<br><br>
>***Example***: 
>>Input: l1 = [2,4,3], l2 = [5,6,4]
>>  <br>
>>Output: [7,0,8]
>>  <br>
>>Explanation: 342 + 465 = 807
>><br>
>>inverse of 807 : 708 >>[7,0,8]
